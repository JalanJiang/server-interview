### 链表

#### 单链表

[搞懂单链表常见面试题](https://juejin.im/post/5aa299c1518825557b4c5806)

特点：

1. 删除元素时间 `O(1)`，查找一个元素复杂度 `O(n)`
2. 不需要预先分配空间大小，避免空间浪费
3. 无法回溯操作

基本操作：

1. 获取链表长度：遍历链表节点 
2. 查询节点
3. 添加元素
4. 删除元素

!> 由单链表的增加删除可以看出，链表的想要对指定索引进行操作（增加，删除）的时候必须获取该索引的前一个元素。
